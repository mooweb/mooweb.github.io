Omaha South Summer 2019 Community Tech Project Pilot

This is a pilot program designed to help students develop professional skills through a collaboration with employees in tech at Mutual of Omaha. Students will work with mentors from Mutual of Omaha to help them develop web based tools/apps to address an issue they see in the community.
